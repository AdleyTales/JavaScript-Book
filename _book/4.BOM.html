4. BOM

BOM

  • BOM: 称为浏览器对象模型。意思就是使用JS脚本，来控制浏览器。
  • 我们知道 window 是浏览器顶层对象。在 BOM 中，所有的属性、方法都是挂在 window 这个对象下。
  //demo1
  var a = 1000;
  console.log(window.a == a); // true

  // demo2
  var b = 200;
  function getNum(){
    console.log(this === window); // true
    return this.b; //此时的this == window
  }
  var num = getNum();
  console.log(num); //200
